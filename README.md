# Calcards
Um jogo para dois jogadores onde vocês competem para obter a expressão matem que gera o maior resultado, com o apoio de números, operadores e truques!
Cada jogador recebe diferentes cartas que possibilitam a geração de expressões matemáticas a fim de gerar um número maior que seu oponente.
No início do jogo, são distribuídas 9 cartas, 6 números e 3 operações, para cada jogador. Ao fim da rodada, os jogadores recebem, cada um, 3 cartas extras: uma operação e dois números.
Mas não para por aí! Os jogadores também recebem cartas chamadas "truques", que alteram as expressões de cada jogador, adicionando outra camada de estratégia na construção da sua jogada!
Assim, o jogador que obtiver o maior valor por meio de sua expressão matemática ganha a rodada, e aquele que ganhar primeiro 7 rodadas ganha o jogo.
Níveis de dificuldade: ao iniciar o jogo, os jogadores podem escolher entre o baralho básico, que oferta operadores simples (+, -, *, /) e o baralho avançado, que oferta operadores como (^, (), !, √) que possibilitam cálculos mais complexos e, consequentemente, números maiores!
Com suporte do OperaGx Games, plataforma parceira do GameMaker, você pode convidar seu amigo para uma partida multiplayer online de forma rápida e prática.
